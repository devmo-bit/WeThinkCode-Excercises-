This repository contains solutions to the WeThinkCode programming exercises.  
Each exercise is implemented in Java and focuses on building fundamental coding skills through practical examples.

📂Contents
- **Exercise 1:** Factorial (Recursion & Code Comprehension)
- **Exercise 2:** Greeter (Documentation & String Handling)
- **Exercise 3:** Sum Example (Debugging)
- **Exercise 4:** Add Function (Unit Testing with JUnit)
- **Exercise 5:** EvenCheck (Refactoring)
- **Exercise 6:** HelloWorld (Java Basics)
- **Task Manager:** A simple console-based task manager to add, view, and complete tasks.

🎯Purpose
The goal of this repository is to:
- Practice core programming concepts (loops, conditionals, recursion, OOP).
- Learn debugging, testing, and refactoring techniques.
- Build confidence in writing clean, maintainable Java code.
